¿Que es?
El modulo "Welcomer" es un modulo que permite crear mensajes de Bienvenidas y Despedidas, eso puedes hacerlo con un mensaje de Texto o un mensaje Embed.

Información
Opcion Estado/Valor Mensaje
Modulo configurable Si Este modulo es configurable.
Configuracion de mensajes No Los mensajes de Logs son predeterminados.
Configuracion de canal Si Si, el modulo te permite elegir un canal.
